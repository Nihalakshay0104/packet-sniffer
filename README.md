# Network Packet Sniffer & Traffic Analyzer

## Features
- Real-time packet capture
- Protocol detection (TCP/UDP)
- Traffic analysis (top IPs, stats)
- Suspicious activity detection
- Export data to CSV

## Technologies
- Python
- Scapy
- Tkinter

## How to Run
```bash
pip install scapy
python advanced_sniffer.py
